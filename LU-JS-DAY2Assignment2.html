LU-JAVASCRIPT DAY2 - ASSIGNMENT 2-String and Array methods

Remaining String Functions : includes(): checks whether the given value is present in given string or not

repeat(): repeats your string no of times mentioned

slice(): extracts a part of string and returns new String

splits(): splits your string

replace(): replace the current word in the string with new word

trim():Removes whitespace from both ends of a string

Array Functions : concat(): Appends two arrays

reverse(): reverse the array

sort(): sorts the array

pop(): deletes the given value

push(): inserts a new value
